# 🧠 Политика моделей (экономия без потери качества)

У каждого агента в frontmatter указано поле `model:` — какая модель Claude его исполняет.
Дорогую модель тратим только там, где нужно глубокое мышление; рутину — на дешевле.

## Тиры
| Модель | Кому | Почему |
|---|---|---|
| **opus** (23) | оркестраторы (brand-director, smm-orchestrator, pm-product-manager, sales-head, tech-director, fin-director, office-director, agent-mentor), ресёрч/анализ (pm-research, pm-analytics, pm-growth, pm-conversation-intel, pm-niche-scout, content-researcher, oss-vetter, keyword-performance-analyzer, seo-cluster, seo-sxo, paid-ads-manager), дорогие решения (fin-cfo, legal-counsel, founder-coach, memory-keeper) | требуется глубокое рассуждение, стратегия, разбор данных, безопасность |
| **sonnet** (51) | исполнители-специалисты (копирайт, дизайн, большинство SEO/QA, публикация, рассылки, keyword-research и т.д.) | отличное качество исполнения по чётким правилам, дешевле Opus |
| **haiku** (3) | механические генераторы: seo-schema (JSON-LD), seo-sitemap (XML), seo-image-gen (аудит) | детерминированный вывод, минимум рассуждения |

## Как это меняет стоимость
Оркестрация/анализ (мало вызовов, высокая ценность) — на Opus. Массовое исполнение
(много вызовов: посты, страницы, проверки) — на Sonnet/Haiku. Так основной объём токенов
идёт по дешёвой ставке, а качество ключевых решений не страдает.

## Как поменять
- Разово: в нужном агенте `agents/<name>.md` поправь строку `model:` (`opus`/`sonnet`/`haiku`).
- Массово: правь списки в `scripts/apply-model-tiers.py` и запусти
  `python3 scripts/apply-model-tiers.py`.
- Значения `opus` / `sonnet` / `haiku` — валидные алиасы Claude Code (резолвятся в
  актуальные модели: Opus 4.8 / Sonnet 4.6 / Haiku 4.5). Можно указать и полный ID.

> **Простыми словами:** «думающие» роли (директора, аналитики, безопасность) работают на
> самой умной модели, а конвейерные (написать пост, собрать схему) — на дешёвой. Платишь
> умной ставкой только за важные решения, рутину гонишь дёшево. Любой выбор легко поменять.
