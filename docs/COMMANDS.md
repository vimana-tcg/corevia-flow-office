# 📖 Команды Corevia Flow Office — какими словами что вызывать

Вы пишете обычными словами — Claude сам подбирает нужного агента/скилл. Ниже —
триггеры (слова-ключи) и что произойдёт. Языки: рус/укр/англ.

> Не нужно запоминать. Просто опишите задачу человеческими словами — сработает.

## 🛠️ Настройка и помощь
| Скажите | Сработает | Результат |
|---|---|---|
| `/setup`, «настройка», «начать», «подключить ключи» | Мастер setup | Профиль + подключение API по шагам |
| `/doctor`, «что подключено», «проверь ключи» | Doctor | Список подключённых/недостающих сервисов |
| `/cases`, «примеры», «кейсы», «как используют» | Cases | Реальные сценарии применения |

## 🎯 Продукт (PM)
«стратегия», «куда двигать продукт», «приоритеты», «roadmap», «A/B тест», «аналитика»,
«почему не покупают», «новая ниша» → PM-команда (стратегия, данные, рост, голос клиента).
