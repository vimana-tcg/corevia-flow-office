# 🏢 Corevia Flow Office — ваша AI-команда в коробке

Готовый набор **AI-сотрудников** для Claude Code: продукт-менеджеры, маркетологи,
дизайнеры, SEO, копирайтеры, финансисты, безопасники и тестировщики. Вы пишете
простыми словами, что нужно — нужный «отдел» берётся за работу.

**Для кого:** соло-фаундеры и малые команды. **Опыта в коде НЕ требуется** — мастер
настройки проведёт за руку.

> ⚠️ Это оркестрация поверх [Claude Code](https://docs.claude.com/claude-code). Вы
> используете СВОИ ключи API (платите напрямую сервисам). Мы ничего не собираем и не
> храним — никакой телеметрии.

---

## 🚀 Быстрый старт (одна команда)

```bash
curl -fsSL https://raw.githubusercontent.com/vimana-tcg/corevia-flow-office/main/scripts/bootstrap.sh | bash
```
Это скачает набор и подключит **всех агентов + все скиллы** к вашему Claude Code.

Или вручную:
```bash
git clone https://github.com/vimana-tcg/corevia-flow-office ~/corevia-flow-office
cd ~/corevia-flow-office && bash scripts/install.sh
```

Дальше — **перезапустите сессию Claude** (чтобы он увидел новых агентов) и напишите:
- **`/tour`** — экскурсия простыми словами: как всё устроено, кто за что, схемы (см. [`docs/GUIDE.md`](docs/GUIDE.md)). Можно спросить «как работает X» и «кто у меня есть».
- **`/setup`** — мастер: спросит про вас (имя/бизнес/город/цель) и по шагам подключит ключи (OpenAI, Replicate, Canva, Meta, YouTube, Pexels…). Ничего знать заранее не нужно.
- **`/doctor`** — что уже подключено. **`bash scripts/setup-video.sh`** — софт для видео (ffmpeg).

Кода знать НЕ нужно — пишите задачи обычными словами.

## 🧩 Что внутри (отделы)

| Отдел | Что делает | Примеры команд |
|---|---|---|
| **Продукт (PM)** | стратегия, приоритеты, аналитика, рост | «куда двигать продукт», «A/B тест» |
| **Маркетинг/SMM** | контент-план, посты, реклама, перформанс | «сделай креативы», «план на неделю» |
| **Дизайн** | лендинги, UI, баннеры, анимация | «сделай красивый лендинг» |
| **SEO** | аудит сайта, контент-кластеры, AI-поиск | «проверь SEO», «audit сайта» |
| **Контент** | статьи, посты, видео-сценарии | «напиши статью», «шортс» |
| **Финансы** | учёт, P&L, runway, юнит-экономика | «сколько денег», «runway» |
| **Безопасность** | проверка кода/пакетов, security-review | «проверь репозиторий», «security review» |
| **QA** | поиск багов, тесты, проверка перед запуском | «найди баги», «протестируй» |

Полный список и **какими словами что вызывается** → [`docs/COMMANDS.md`](docs/COMMANDS.md).

## 🔄 Обновления
Мы регулярно улучшаем команды. Чтобы получать новое:
```bash
bash scripts/update.sh      # подтянет последнюю версию (git pull upstream)
```
Нажмите на GitHub **Watch → Custom → Releases** — будут приходить уведомления о
новых версиях. История изменений — [`CHANGELOG.md`](CHANGELOG.md).

## 🔐 Безопасность и приватность
- Ваши ключи живут в `.env` — он **не** попадает в git.
- Перед каждым коммитом работает **секрет-страж** (`scripts/scan-secrets.sh`).
- Нет телеметрии. Вы сами отвечаете за расходы на API и за соблюдение правил площадок.
