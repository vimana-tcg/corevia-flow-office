# Кейс 001 — 5 рекламных креативов из одного оффера

**Задача:** из одного предложения сделать пачку разных рекламных картинок для теста.
**Кто:** соло-фаундер, ноль опыта в дизайне.
**Команды:** `ad-creative-studio` (+ `/setup` для ключей Replicate/Pexels).

## Как было сделано
1. `/setup` → подключили Replicate (картинки) и Pexels (бесплатные фото). 5 минут.
2. Написали: «сделай 5 рекламных креативов под оффер: <ваш оффер>».
3. Скилл взял фон (стоковое фото / ИИ) и наложил текст НАСТОЯЩИМ шрифтом (буквы чёткие,
   ИИ-текст не используется — он ломается), выдал 5 РАЗНЫХ углов: боль / доказательство /
   оффер / сравнение / интрига — с разными цветами и кнопками.
4. Выбрали победителя, остальные — в A/B-тест.

## Ключевой приём (почему работает)
ИИ рисует только КРАСИВЫЙ ФОН; заголовок и кнопку «печатаем» кодом настоящим шрифтом →
идеальный текст на любом языке. Один оффер → 5 заходов = есть что тестировать.

## Что получили
5 готовых креативов (1:1 и 9:16) за ~10 минут, ~$0.15 на картинки. Текст без ошибок.
