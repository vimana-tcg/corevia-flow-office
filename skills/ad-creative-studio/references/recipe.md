# Recipe — почему гибрид и как выбирать движок (заметки, читаются по требованию)

## Доказанный факт
- **Ideogram v3** (ИИ рисует и текст): картинка 🔥, но кириллический текст ломается
  («ЭКСПОРТ ПОД КЛЮЧ» → «ЭКСРОРТ РОД КЛОЧ»; «КОНТРАКТОВ» → «КОНТРРАКТЫВ»).
- **Gemini/nano-banana**: лучший текст из ИИ, но требует платный ключ (free-tier
  отдаёт 429).
- **Canva**: текст идеальный, но визуал «шаблонный» + ручная работа.
- **Вывод:** ИИ/сток → ФОН; текст → код. Идеальная кириллица + любой язык сменой строки.

## Какой фон брать
| Кейс | Движок | Почему |
|---|---|---|
| Дефолт, доверие, B2B | **Pexels stock** (бесплатно) | реальные фото = доверие |
| Нужна уникальная сцена/арт | **Ideogram** (Replicate, ~$0.03) | БЕЗ текста в промте, оставить место |
| Реальный товар/услуга | фото бизнеса + базовая ретушь | честность |
| Лицо/портрет на обложке | rembg-вырезка → поле `face` | см. SKILL.md, шаг 3.5 |

## Стиль текста
- **clean** — премиум-минимализм: Montserrat Black заголовок, eyebrow SemiBold +
  акцентная линия, body Medium, акцентная CTA. Для B2B-бренда, лендингов, серьёзных офферов.
- **mrbeast** — Montserrat Black + чёрная обводка (`stroke_width ≈ hsize*0.07`) +
  `accent_word` цветом + UPPERCASE. Текст читается на ЛЮБОМ фоне, «кликбейт». Для верха
  воронки, охватных Reels/Shorts-обложек, TikTok.

## Шрифты (assets/fonts/)
- **Montserrat** — премиум-гротеск, Cyrillic, веса Thin…Black (вариативный). Дефолт.
- Можно добавить любой бренд-шрифт проекта рядом и переключить путь FONT в overlay.py.
- Грузить вес через `font.set_variation_by_name('Black'|'ExtraBold'|'SemiBold'|'Medium')`.

## Бренд-дефолты (НАСТРАИВАЕМЫЕ — поменяй под свой бренд)
- Пример тёмной премиум-палитры: фон тёмный, акцент золото `#F0B223`, текст белый,
  CTA-pill в цвет акцента. Это один из вариантов — палитра задаётся полем `accent_color`
  (см. `references/angles.md`) или константами в overlay.py.
- Бренд-лок (название бренда) сверху-слева, разрядка букв — поле `brand` в конфиге.
- Форматы: 1:1 (лента), 9:16 (Reels/Stories), 16:9 (YouTube/landscape).
