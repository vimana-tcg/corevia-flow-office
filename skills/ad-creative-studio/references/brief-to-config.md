# Brief → creatives.json (схема конфига для overlay.py)

Из брифа (концепт + копи + визуал-дирекшн) собирается JSON. Пример на 2 креатива:

```json
{
  "brand": "ACME",
  "out_dir": "output/creatives/final",
  "style": "clean",
  "creatives": [
    {
      "id": 1,
      "size": [1080, 1080],
      "bg": "output/bg/bg_1.jpg",
      "eyebrow": "ДЛЯ МАЛОГО БИЗНЕСА",
      "headline": "Сервис под ключ",
      "accent": "$29 / месяц",
      "sub": "Настройка · поддержка · аналитика — всё включено",
      "cta": "Бесплатный разбор →",
      "hsize": 96, "ssize": 36
    },
    {
      "id": 4,
      "size": [1080, 1920],
      "bg": "output/bg/bg_4.jpg",
      "eyebrow": "ТРАНСФОРМАЦИЯ",
      "headline": "ЗАПУСК — это ПРОСТО",
      "accent_word": "ПРОСТО",
      "sub": "От идеи → первые клиенты за неделю",
      "cta": "Начать →",
      "hsize": 92, "ssize": 38, "style": "mrbeast"
    }
  ]
}
```

## Поля
| Поле | Обяз. | Значение |
|---|---|---|
| `id` | да | имя файла → `final_<id>.png` |
| `size` | да | `[1080,1080]` (1:1) · `[1080,1920]` (9:16) · `[1920,1080]` (16:9) |
| `bg` | да | путь к фону (стоковое фото / ИИ-фон БЕЗ текста) |
| `eyebrow` | нет | мелкая надпись над заголовком (акцент, разрядка) + акцентная линия |
| `headline` | да | главный заголовок (перенос авто) |
| `accent` | нет | строка-акцент под заголовком (акцент, крупно) — цена/оффер |
| `accent_word` | нет | слово внутри headline, которое красим акцентом |
| `sub` | нет | подпись (серый, Medium) |
| `cta` | нет | текст кнопки (pill в цвет акцента) |
| `hsize`/`ssize` | нет | кегль заголовка/подписи (дефолт 88/36) |
| `style` | нет | `clean` (дефолт) · `mrbeast` (обводка + UPPERCASE + accent_word) |
| `face` | нет | путь к PNG-вырезу лица (любой rembg-инструмент, см. SKILL.md шаг 3.5) |
| `face_side` | нет | `right` (дефолт) · `left` — куда поставить лицо |
| `face_scale` | нет | доля высоты под лицо (дефолт 0.92) |

## Правило размеров (чтобы влезало)
- Длинный headline → меньше hsize (74–80). Короткий хук → 96–110.
- mrbeast обычно UPPERCASE + короче (хук в 2-4 слова).
